Detailed Description

template<typename _Tp>
class zl::Point_< _Tp >

template 2D point class.

The class defines a point in 2D space. Data type of the point coordinates is specified as a template parameter. There are a few shorter aliases available for user convenience. See zl::Point, zl::Point2i, zl::Point2f and zl::Point2d.
